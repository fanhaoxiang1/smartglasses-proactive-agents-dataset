# Dataset for "Toward Personalized Proactive Agents on Smart Glasses"

> **Status:** This repository is currently **private / anonymized** for peer review.
> It will be made public upon paper acceptance.

This repository hosts the dataset and accompanying documentation for the paper:

## Overview
The dataset links **user profiles → personalized contexts/policies → perceptions of proactive behaviors** in smart-glasses-like everyday scenarios.
It was collected through **structured video elicitation workshops** with **21 participants** across **18 first-person scenarios**.
