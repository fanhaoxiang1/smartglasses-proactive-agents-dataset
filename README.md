## Overview
This dataset supports the study *“Toward Personalized Proactive Agents on Smart Glasses: From Explicit Context Policies to Implicit User Differences.”* It links **user profiles → personalized contexts/policies → perceptions of proactive behaviors** in smart-glasses-like everyday scenarios. The dataset was collected through structured video elicitation workshops with 21 participants across 18 first-person scenarios.

Our work investigates how personalization can improve proactive agents by bridging **explicit policy specification** (e.g., user-defined context rules) and **implicit user differences** (e.g., personality, autonomy preferences).
